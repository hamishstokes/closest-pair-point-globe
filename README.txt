Finds the closest pair of points on the earth given a list of points defined by latitude and longitude
Input from stdin

Input example:
3								//3 points to be compared
Berlin 52.523405 13.4114		//Name followed by latitude then longitude
Barcelona 41.387917 2.169919
Dresden 51.050991 13.733634
2								//second set of points
Berlin 52.523405 13.4114
Barcelona 41.387917 2.169919
0


Output produced:
Scenario 1:
Closest pair: Berlin Dresden	//Outputs closest pair
Distance: 165.2					//gives distance between pair
Scenario 2:
Closest pair: Barcelona Berlin
Distance: 1499.8
